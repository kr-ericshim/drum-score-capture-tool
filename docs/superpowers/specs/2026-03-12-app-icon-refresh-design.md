# App Icon Refresh Design

## Goal

Make the app icon read as a focused desktop productivity tool while still clearly signaling the app's actual job: capturing score pages from video.

## Approved Direction

- Primary metaphor: large score page
- Secondary metaphor: capture frame corners
- Avoid: checklist panel, generic task manager symbols
- Tone: bright background with a sharp accent

## Visual Structure

The icon should use a bright rounded-square tile as the base. A large score page should dominate the center so the icon reads as "sheet music" first. Capture-frame corners should sit around the page as a supporting cue that the page is being extracted or cropped from video, not merely viewed.

## Styling

- Base tile: light cream or cool white gradient
- Main lines: deep navy/slate for clarity at small sizes
- Accent: one warm highlight for capture corners
- Detail budget: minimal, optimized for dock/taskbar readability

## Success Criteria

- At small sizes, the page silhouette remains the strongest shape
- The icon no longer looks like a generic checklist or notes app
- The capture metaphor is visible without overpowering the score-page metaphor
